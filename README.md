# Methane Justice: Satellite Data Analysis

## Overview
Who emits methane? Who suffers climate impacts? This project visualizes the climate injustice.

## Data
- **EDGAR**: Global methane emissions by country (2018)
- **ND-GAIN**: Climate vulnerability index by country
- **TROPOMI**: Satellite methane observations (2017-present)

## Key Finding
- China, India, USA, Brazil, Russia = 73% of global methane emissions
- These countries have LOW climate vulnerability (avg 35/100)
- Most vulnerable countries emit almost nothing
- **This is climate injustice**
